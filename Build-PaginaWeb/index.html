Como Jugar

Utilizando las flechas o las teclas WASD mueve el personaje para aprender acerca de los componentes principales de las computadoras y ayuda a salvarla de virus !.
Resuelve los minijuegos y aprende acerca de los diferentes componentes de las computadoras.
Ve a el edificio central para terminar o salir del Juego.

Trailer

Acerca del Proyecto

Este proyecto fue creado con el proposito de promover las carreras relacionadas a STEAM (Science, Technology, Engineering, Art, Mathematics) y enseñar de una manera basica los componentes de la computadora.
Fue creado con el Unity Game Engine version 2019.4.25f1 LTS.

¿ Como puedo yo hacer mis propios juegos? A : Un buen lugar para empezar es utilizando un Game Engine, el cual te permite crear juegos de una manera mas rapida y divertida, un buen lugar para empezar es aqui --> Unity Learn
